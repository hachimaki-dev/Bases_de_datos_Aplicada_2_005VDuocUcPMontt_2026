Descodificando SQL

Ver una consulta gigante te vuela la cabeza porque intentas leerla de arriba a abajo. Para entender SQL, debes separar "qué hace" de "cómo se escribe".

Desbloquearemos el código paso a paso superando pequeños desafíos de lógica. ¡Comencemos!

Fase 1

El Catálogo (FROM)

SQL parece escribir el SELECT al principio, pero en su mente, lo primero que lee es el FROM. No puedes organizar un inventario de películas si primero no decides a qué bodega o catálogo vas a entrar.
FROM contenidos

Reto para avanzar:

Si tuvieras que organizar un reporte de inventario en tu plataforma, ¿a qué lugar físico corresponde el FROM?

Fase 2

Los Estantes (GROUP BY)

Una vez dentro de la bodega, sacar miles de cintas sueltas es caótico. El segundo paso mental de SQL es acomodarlas en Estantes (Agrupar). Toma las películas sueltas y las organiza en pasillos separados por género (El pasillo de Terror, el de Comedia).
FROM contenidos
GROUP BY genero

Verdadero o Falso:

"Si mi tabla tiene 1,000 películas, pero solo existen 5 géneros distintos. Al hacer GROUP BY genero, la tabla resultante se encogerá a mostrar como máximo 5 grandes estantes (que representan 5 filas de resultados)."

Fase 3

El Reporte de Inventario (SELECT)

Finalmente imprimimos el reporte. El SELECT es tu libreta de apuntes cuando caminas por los estantes: sólo te enseña las etiquetas de las repisas y aplica matemáticas hacia adentro de cada estante (contando, sumando).
SELECT genero,
       COUNT(*),
       SUM(duracion_min)
FROM contenidos
GROUP BY genero;

Traduce para la Jefatura:

¿Qué significa exactamente escribir COUNT(*) versus SUM(duracion_min) en el ticket?

Fase 4

La Cebolla (Funciones)

SQL permite anidar funciones (poner una dentro de otra). Como una cebolla o una muñeca rusa, el cerebro debe leerlas de ADENTRO hacia AFUERA.
ROUND( AVG(rating), 1 )
1º Capa Interna -> Promedio 2º Capa Externa -> Redondear

Aplica la regla:

Si necesitas convertir a MAYÚSCULAS el recorte de las 3 primeras letras de un Nombre... ¿Cuál sintaxis anidada lo lee "de adentro hacia afuera"?

Batalla Final: La Consulta Completa

Has superado todas las fases. Ya no deberías sentir pánico al ver todo combinado. Demuestra que puedes traducirlo.

labestia.sql
SELECT genero, COUNT(*) AS cantidad, ROUND(AVG(rating), 1) AS rating_promedio, SUM(duracion_min) AS minutos_totales FROM contenidos GROUP BY genero;

¿Cuál es la traducción exacta en lenguaje humano?
